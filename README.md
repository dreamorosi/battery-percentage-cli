# battery-percentage-cli

> Get current MacOS battery level

Works on macOS only.

## Install
```
$ npm install --global battery-percentage-cli
```

## Usage
```
$ battery-percentage --help

  Get current MacOS battery level

  Usage
    $ battery-percentage

  Options
    -v, --verbose     Get all info

  Example
    $ battery-percentage
    36%
```
